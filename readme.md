# Multi-step Form

## Description
> Implémentation d'un formulaire multi-step relié à une base de donnée. 

1. Test de la mise en forme avec TailWind
2. Test requêtes & base de donnée avec Postman
3. Mise en place d'une DB sql
4. Refacto -> React
